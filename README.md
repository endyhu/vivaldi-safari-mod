# vivaldi-safari-mod

A CSS mod that gives [Vivaldi Browser](https://vivaldi.com) a Safari-inspired look and feel.

## ⚙️ Required Vivaldi Settings

Before applying the mod, configure Vivaldi as follows:

### Tab Bar Position → Bottom

1. Open **Settings**
2. Go to **Tabs**
3. Under **Tab Bar Position**, select **Bottom**

### Show Popup Thumbnails → Off _(recommended)_

1. In **Settings**, go to **Tabs**
2. Disable **Show Popup Thumbnails**

> **Why?** The tab thumbnail popups currently appear upward, which looks off when the tab bar is at the bottom. They don't yet snap downward as you'd expect. Turning them off avoids this visual glitch.
>
> 🔧 _This is a known issue - ideally the popup would appear above the content (i.e. downward from the tab bar's perspective). A fix may come in a future update._
